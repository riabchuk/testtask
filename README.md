<p align="center">
Встановлення

Для розгортання проекту достатньо скопіювати файл конфігурації
<code> cp .env.example .env </code>
та запустити команду
<code> make setup </code>
</p>
<p> після цього проект буде доступний за адресою <a href="http://localhost:8090/">http://localhost:8090/</a>

<p>Для тестування API можна використати колекцію testtask.postman_collection.json що знаходиться в корені проекту</p>
